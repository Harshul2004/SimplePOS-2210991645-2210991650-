# SimplePOS

# Features

🔥 Invoice Generation.

⚡ Automated Sales Performance Metrics.

🧠 Real-time Sales.

🎯 Digital Receipt Generation, Customizable Product listings, and Support for Multiple Printers.

# 🛠️ Tech Stack

Frontend:
WinForms, WPF technologies.

Backend:
C#, SQLite database.
